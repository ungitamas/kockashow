# KockaShow weboldal

Ez a projekt egy egyszerű, HTML és CSS alapú bemutató weboldal, amely a KockaShow témájára készült.

Az oldal alapja egy W3Schools W3.CSS sablon, amelyet átalakítottam egy LEGO kiállítást és családi élményprogramot bemutató weboldallá.

## A projekt célja

A feladat célja, hogy gyakoroljam:

- a HTML fájl felépítését,
- a CSS használatát,
- egy meglévő sablon átalakítását,
- képek és szövegek cseréjét,
- GitHub repository használatát,
- commitok készítését.

## Használt technológiák

- HTML
- CSS
- W3.CSS
- Font Awesome ikonok

## Forrás

Az oldal alapjául a W3Schools egyik W3.CSS sablonja szolgált:

[W3.CSS Templates](https://www.w3schools.com/w3css/w3css_templates.asp)

A választott sablon: **Band Template**

## Saját módosítások

A sablonban több részt is átalakítottam:

- a képeket KockaShow témájú képekre cseréltem,
- az angol szövegeket magyar szövegekre cseréltem,
- a zenekaros témát LEGO kiállítás témára alakítottam,
- a közösségi média ikonokat webfejlesztéshez kapcsolódó ikonokra cseréltem,
- egyedi `style.css` fájlt kapcsoltam az oldalhoz.

## Fájlok

- `index.html` – az oldal fő HTML fájlja
- `style.css` – saját stílusok
- `README.md` – a projekt leírása
- `img/` – a weboldalon használt képek mappája
